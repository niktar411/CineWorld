Документация по запуску проекта CineWorld Registration
Описание проекта
Проект представляет собой адаптивную веб-форму регистрации для кинопортала CineWorld с поддержкой темной/светлой темы и валидацией данных.

Структура проекта
text
project-folder/
├── index.html              # Основная страница регистрации
├── success.html            # Страница успешной регистрации
├── globals.css             # Глобальные стили и сброс стилей
├── styleguide.css          # CSS-переменные и темы
├── style.css               # Основные стили компонентов
├── body.css                # Структурные стили страницы
├── mobile.css              # Адаптивные стили для мобильных устройств
├── theme.js                # Логика переключения тем
├── script.js               # Функционал календаря
├── register.js             # Валидация формы регистрации
└── img/                    # Папка с изображениями
    ├── icon.svg            # Логотип (светлая тема)
    ├── icon_black.svg      # Логотип (темная тема)
    ├── divider.svg         # Разделитель в хедере
    ├── right.svg           # Стрелка календаря (светлая тема)
    └── right_black.svg     # Стрелка календаря (темная тема)
Требования
Веб-браузер с поддержкой современных стандартов HTML5, CSS3 и ES6+

Доступ к интернету (для загрузки шрифта Space Mono)

Инструкция по запуску
1. Подготовка файлов
Создайте папку для проекта

Сохраните все предоставленные файлы в этой папке

Создайте подпапку img и поместите в нее все изображения

2. Запуск проекта
Откройте файл index.html в веб-браузере одним из способов:

Вариант A: Двойной клик

Найдите файл index.html в проводнике

Дважды щелкните по нему для открытия в браузере по умолчанию

Вариант B: Через веб-сервер (рекомендуется)
Для корректной работы некоторых функций рекомендуется использовать локальный сервер:

bash
# Если установлен Python
python -m http.server 8000

# Если установлен Node.js
npx http-server

# Если установлен PHP
php -S localhost:8000
Затем откройте в браузере: http://localhost:8000

3. Функциональность проекта
Основные возможности:
Регистрационная форма с полями:

Имя и фамилия (обязательные)

Дата рождения (необязательное)

Email (обязательное)

Пароль и подтверждение пароля (обязательные)

Выбор любимого жанра

Чекбоксы согласия

Валидация данных:

Проверка формата email

Минимальная длина пароля (6 символов)

Совпадение паролей

Проверка даты на корректность

Обязательное принятие условий

Календарь для выбора даты рождения:

Открывается по клику на поле даты

Навигация с ускоренной прокруткой

Невозможность выбрать будущую дату

Переключение темы:

Светлая/темная тема

Сохранение выбора в localStorage

Автоматическое применение при загрузке

Адаптивный дизайн:

Оптимизация для мобильных устройств

Гибкая верстка для разных размеров экрана

4. Особенности реализации
CSS архитектура:
styleguide.css - система дизайна с CSS-переменными

body.css - структурные стили макета

style.css - стили компонентов и анимации

mobile.css - адаптивные медиа-запросы

globals.css - глобальные стили и сброс

JavaScript модули:
theme.js - управление темами и UI-эффектами

script.js - функционал календаря

register.js - валидация формы и обработка отправки

Firefox 55+

Safari 12+

Edge 79+

6. Возможные проблемы и решения
Проблема: Изображения не загружаются
Решение: Убедитесь, что папка img содержит все необходимые файлы

Проблема: Шрифты не подгружаются
Решение: Проверьте подключение к интернету

Проблема: Календарь не открывается
Решение: Запустите проект через локальный сервер (не файловую систему)

Проблема: Тема не сохраняется
Решение: Проверьте, не отключен ли localStorage в браузере

7. Дальнейшая разработка
Для расширения функциональности можно:

Добавить бэкенд для обработки формы

Реализовать отправку данных на сервер

Добавить дополнительные поля валидации

Интегрировать с API кинопортала

Проект готов к использованию и может быть развернут на любом веб-хостинге.
